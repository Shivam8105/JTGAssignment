# Portfolio Website

A modern, responsive portfolio website for Rayan Aldrdard - Front-end Developer.

## Features

- Hero section
- Project showcase with slider
- Smooth animations and transitions

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins)

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Styling and layout
- `script.js` - Interactive functionality
- `images/` - Image assets
